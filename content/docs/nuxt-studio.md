# Ghi chú nhanh Nuxt Studio

Nội dung này chỉ để **tham khảo trong repo**; bạn có thể xóa hoặc sửa tùy ý.

## Trong môi trường dev (`npm run dev`)

1. Mở site (thường là `http://localhost:3000`).
2. Bấm **nút Studio** ở góc dưới bên trái (hoặc vào `/_studio` nếu đã bật route mặc định).
3. Chọn trang hoặc file Markdown cần sửa — editor mở trực tiếp trên preview.
4. **Lưu** sẽ ghi xuống đĩa trong thư mục `content/` (đồng bộ với Git / IDE của bạn).

## Production

- Cần cấu hình **repository** (GitHub/GitLab) và **OAuth** (hoặc token) để đăng nhập và **publish** commit lên remote.
- Chi tiết: [nuxt.studio/setup](https://nuxt.studio/setup) và [Auth providers](https://nuxt.studio/auth-providers).

## Phím tắt (thường gặp)

- **`Cmd + .`** (macOS) — mở Studio nhanh (theo tài liệu Studio).
