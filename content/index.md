# Chào mừng

Đây là trang **Markdown** trong thư mục `content/`. Chạy `npm run dev`, bấm nút Studio góc dưới bên trái để chỉnh sửa trực quan.

## Các trang mẫu

- [Giới thiệu](/about) — `content/about.md`
- [Bài blog mẫu](/blog/bai-mau) — nhiều mục, có mục lục bên phải
- [Demo mục lục đa cấp](/blog/demo-muc-luc-da-cap) — `##` + `###` lồng nhau
- [Ghi chú Nuxt Studio](/docs/nuxt-studio) — `content/docs/nuxt-studio.md`

## Bước tiếp theo

- Thêm file `.md` trong `content/`; URL = đường dẫn file (bỏ `.md`, giữ cấu trúc thư mục).
- Khi deploy production, cấu hình `studio.repository` và OAuth để publish qua Git.
